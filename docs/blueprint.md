# **App Name**: Work Showcase

## Core Features:

- Project Display: Showcase various projects with descriptions and images.
- Contact Form: Allow visitors to send inquiries or messages.
- About Section: Provide information about the work and background.
- Testimonial Carousel: Display client testimonials in an engaging manner, powered by an AI tool to select the most relevant testimonials.
- Blog Section: Host a blog for sharing updates and news related to the field.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3192) for a professional and trustworthy feel.
- Background color: Light grayish-blue (#D1D5DB) for a clean and modern canvas.
- Accent color: Soft lavender (#A78BFA) to add a touch of creativity and highlight key sections.
- Headline font: 'Belleza' (sans-serif) for a fashionable, artistic feel.
- Body font: 'Alegreya' (serif) for readable text.
- Use simple, line-based icons to complement the clean design.
- Implement a grid-based layout with plenty of white space for easy navigation and readability.
- Incorporate subtle transitions and animations on hover to add visual interest without being distracting.
